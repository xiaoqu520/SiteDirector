# 🚀 快速加入

{% hint style="success" %}
**快速入口**：运行 `站点主管-收容协议 启动游戏.exe` → **开始新游戏** → 完成 12 步教程
{% endhint %}

> **[待补图 IMG-002]** 主菜单全屏截图

## 👤 适合谁

第一次在 Windows 或 Android 上启动《站点主管：收容协议》的玩家。

## 📖 简介

从下载到进入站点约 **5～15 分钟**（视安装与网络速度）。本页是「一、开始游戏」的速查入口；细节见 [安装与更新](02-getting-started/install.md)。

## 🛤️ 首启时间线

| 步骤 | 做什么 |
|------|--------|
| 1️⃣ | 运行 **`站点主管-收容协议 SiteDirector-v1.6.1-setup.exe`**（新玩家）或启动已安装 exe |
| 2️⃣ | 等待更新检查完成（已安装用户可能自动下载 ZIP） |
| 3️⃣ | 主菜单 → **开始新游戏** |
| 4️⃣ | 跟随 **12 步新手教程**（约 3 分钟） |
| 5️⃣ | `Ctrl+S` 快速存档 |
| 6️⃣ | 阅读 [第一天生存指南](03-tutorial/first-day.md) |

## 🖥️ 平台速查

| 平台 | 入口 |
|------|------|
| **Windows** | setup.exe 安装 → `启动游戏.exe` |
| **Android** | APK 安装 → 横屏启动 → 见 [移动端](02-getting-started/mobile.md) |

## 🔧 启动失败？

| 现象 | 建议 |
|------|------|
| 无法初始化音频 | 启用系统默认播放设备 |
| 更新失败 | 重新下载 setup.exe |
| Android 签名不一致 | 卸载旧版后重装正式 APK |
