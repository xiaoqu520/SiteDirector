## 异常与收容

| 术语 | 解释 |
|------|------|
| SCP | 需收容的异常个体或物体 |
| Safe / Euclid / Keter | 对象分级：易收容 / 难预测 / 极难收容 |
| 威胁等级 | 1–10 数值，影响猎杀性与事件 |
| 收容等级 | 房间能力 1–4，须 ≥ SCP 需求 |
| 收容失效 / Breach | SCP 脱离单元失控 |
| 重收容 | 将 loose SCP 押回收容单元 |
| 收容措施 | 针对特定 SCP 的额外规程（观察笼、头套等） |
| 临时收容间 | 押送中转，约 20 游戏分钟倒计时 |
| 观察岗 | SCP-173 等需不间断视线的研究员轮班 |
| 外勤上报 | 特工发现新异常后发送至收容面板 |
| GOC 锁 | 70 日未收容 → 全球超自然联盟冻结捕获权 |

## 人员

| 术语 | 解释 |
|------|------|
| 编内人员 | 科研/安保/工程师/医护，有生命与士气 |
| D 级人员 | 高风险实验用，不算编内，有伦理代价 |
| 指派岗位 | v1.6.0+ 将人员设为某房间常驻 |
| 士气 | 影响效率，SCP-999 等可提升 |
| 避难所 | 危机时人员自动寻路的安全房间 |

## 系统与协议

| 术语 | 解释 |
|------|------|
| C.A.S.S.I.E | 站内自主响应 AI |
| MTF | 机动特遣队，捕获 / 紧急召回 |
| O5 理事会 | 基金会最高权力机构 |
| O5 合同 | 可选长期任务，有奖励与罚金 |
| 审计评级 | 0–100，影响拨款、MTF 费用、breach RNG |
| 威胁等级 | 1–10，站点整体危险程度 |
| 运营评分 | 审计/电力/收容/连通/后勤/威胁加权分 |
| 毁灭协议 | 审计极低 + 多 breach → 30 分钟核弹倒计时 |
| O5 齐射 | 多弹头联合打击，682 等需此方式 |
| 负载削减 | 电力不足时按优先级断电 |
| GATE A/B/C/D | 站点入口闸口，A 为地表主入口 |
| 检查点 | 区域交界双扇滑动门 |

## 科研

| 术语 | 解释 |
|------|------|
| 科研中心 | 全站限 1，解锁研究树 |
| 科研实验室 | 全站限 2，各 +1 并行槽位 |
| 特性 / 材料 / 规程 | 每个 SCP 的三条科研链 |
| 研究里程碑 | Safe 150/400/1000 · Euclid 200/600/1500 · Keter 300/900/2000 |
| 手动重收容计数 | 累计 10 次可解锁收容措施（无需规程科研） |
