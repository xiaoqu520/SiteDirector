# 封锁与 MTF 调度

> **v1.9.0** · MTF 由**站点自养编队**执行。须就绪编队、指挥室与任务补给费；**无外部总部后备**。

---

## 全站封锁

（与 v1.8 相同：检查点关闭、非战斗人员避难、GATE C 允许 MTF 出动。）

---

## MTF 调度（v1.9.0）

| 类型 | 机制 |
|------|------|
| **外勤捕获** | 抽象派遣；编队就绪后即时结算（物资 / 捕获 / 失败） |
| **紧急召回** | **地图实体**入场；接触 loose SCP 后结算重收容或场域压制 |
| **编队切换** | C.A.S.S.I.E →「切换派遣编队」「切换召回编队」 |

### 前提

| 条件 | 说明 |
|------|------|
| 就绪编队 | 人数达标、就绪度 ≥ 阈值、非恢复中 |
| 指挥室 | 通电 |
| 外勤捕获 | 材料节点已解锁；封锁期须 GATE C |
| 紧急召回 | breach 升级 **阶段 ≥3**；无进行中 MTF 现场行动 |

### 费用

任务**补给费** ≈ ¥20,000 + ¥2,500×队员数（非旧版 ¥150,000 总部费）。

停机坪缩短编队**恢复时间**（不再提高费用）。
