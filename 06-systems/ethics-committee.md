# 伦理委员会

> **v1.8.0** · 基金会内部道德监督机构。平衡站点行动的「道德成本」，防止基金会在收容过程中自我腐化。

---

## 设定依据

玩法设计参考 SCP 官方文档 [Ethics Committee Orientation](https://scp-wiki.wikidot.com/ethics-committee-orientation)（Voct，CC BY-SA）：

- 委员会向 O5 建议何为**可接受**行为
- 调查、传唤、质询 — 表面温和，忽视后果严重
- 反对**不必要的 cruelty**；裁定何时 cruelty **必要**
- 「Protect」— 保护人类免受 SCP 威胁，也保护基金会免受自身

## 部门面板（伦理 Tab）

| 区域 | 功能 |
|------|------|
| **合规概览** | 审计分、训诫等级、调查期 / 基金会审查状态 |
| **D 级申请** | 未批准时提交伦理申请；批准后在此开关劳务 |
| **待办审查案** | 点击案件查看详情并选择回应 |
| **最近结案** | 只读历史（最近 5 条） |

移动端：「更多」菜单 → **伦理**。

---

## 审查案类型

| 类型 | 触发 | 典型回应 |
|------|------|----------|
| **D 级启用申请** | 玩家提交 D 级伦理申请 | 提交规程 / 最小化方案 / 强行启用 |
| **损耗报告** | D 级月度测试损耗 | 提交报告 / 修订规程 / 异议 |
| **突破问责** | SCP 突破升级至阶段 2 | 说明合规 / 承认失职 / 异议 |
| **例行审查** | 每 60 日或低审计 | 配合审查 / 申请延期 |
| **研究监督** | 完成含伦理审查设施的研究 | 批准监督 / 暂停研究 / 异议 |

**逾期未回应** 视为不配合 → 训诫升级 + 伦理调查期。

---

## 训诫与调查

| 训诫等级 | 含义 |
|----------|------|
| 0 | 无 |
| 1 | 书面警告 |
| 2 | 拨款关注 |
| 3 | 严重训诫 |

- **伦理调查期**（7 天）：研究产出 ×0.85，可与基金会审查 -8% 叠加
- 训诫 ≥3 且审计 < 30 → 触发 **O5 基金会审查**

## 伦理审查办公室

- 房间 ID：`room.admin.ethics`
- **全站唯一**，建于行政办公区
- **通电** 时：审查案期限 +2 天，训诫升级阈值提高
