# 👣 12 步完整 Walkthrough

> 每步含：**目标 · 操作详解 · 为什么要做 · 常见错误 · 下一步建议**

---

## 步骤 1 · 欢迎，站点主管

| | |
|---|---|
| **阶段** | 认识界面 |
| **目标** | 点击「下一步」 |
| **自动动作** | 打开简报面板（v1.6.1 桌面端修复） |

**详解**

O5 理事会任命你负责 SCP-CN-465 日常运转。教程约 3 分钟，随时可在设置重看。

**常见错误** — 急着点下一步却跳过任命函。任命函里有初始任务语境。

**下一步** → 步骤 2：读 O5 任命函

---

## 步骤 2 · 作战简报

| | |
|---|---|
| **目标** | 打开 O5 任命函并关闭邮件窗口 |
| **触发** | `TutorialTrigger.MailOpened` |

**详解**

* **简报** 面板 = 站点态势中心
* 下方 **通讯区** = 邮件列表（新邮件置顶，v1.6.0+）
* 点击标题 → 全屏邮件阅读器

**为什么要做** — 邮件是 O5 合同、财政预警、超期催办的主要渠道。养成每日扫邮件的习惯。

**常见错误** — 只关面板没点开邮件。必须 **打开** 邮件才触发。

**下一步** → 步骤 3：熟悉地图

---

## 步骤 3 · 地图浏览

| | |
|---|---|
| **目标** | 缩放或平移地图 |

**详解**

| 操作 | 方式 |
|------|------|
| 缩放 | 滚轮 |
| 平移 | 中键拖动 或 WASD |
| 选中 | 左键 → 左下详情 |
| 切层 | 底部 地表/中层/深层 |

**常见错误** — 面板打开时滚轮无效。先点一下地图区域。

**下一步** → 步骤 4：学习暂停

---

## 步骤 4 · 时间与暂停 ⚠️ 强制

| | |
|---|---|
| **目标** | 按空格暂停或继续 **一次** |
| **不可跳过** | `AllowManualAdvance = false` |

**详解**

顶栏显示：运行天数 · 余额 · 电力 · 威胁 · 审计。

**为什么要做** — 暂停是主管最强大的工具。建造规划、读邮件、应对 breach 都应先暂停。

**常见错误** — 连按两次空格以为坏了。只需切换一次即可。

**下一步** → 步骤 5：财政

---

## 步骤 5 · 财政管理

| | |
|---|---|
| **目标** | 打开「财政」标签 |

**详解**

* 账户余额（开局 ¥500,000）
* 月度收支预估
* 物资库存：清洁水 / 口粮 / 补给

**主管手记** — 建造、招聘、breach 罚款都从这里扣。余额 < −¥100,000 = Game Over。

**下一步** → 步骤 6：建造

---

## 步骤 6 · 网格建造 ⚠️ 强制

| | |
|---|---|
| **目标** | 在地图成功放置一段走廊 |
| **自动动作** | 打开建造页并预选标准走廊 |

**详解**

1. 已选中 **标准走廊**（¥500，工时 240 分钟）
2. 地图左键放置，`R` 旋转
3. 工程师到达后开始施工

**常见错误**

| 错误 | 后果 |
|------|------|
| 放在已有建筑上 | 无法放置 |
| 未连通主廊 | 放置了但无意义 |
| 不等待工程师 | 以为放置即完工 |

**下一步** → 步骤 7：人事

---

## 步骤 7 · 人员编制

| | |
|---|---|
| **目标** | 打开「人事」标签 |

**详解**

五类人员：科研 / 安保 / 工程师 / 医护 / D 级。

右键人员 → 移动或解雇。v1.6.0+ 可 **指派岗位**。

**下一步** → 步骤 8：收容

---

## 步骤 8 · SCP 收容

| | |
|---|---|
| **目标** | 打开「收容」标签 |

**详解**

* 外勤上报的新 SCP
* 已收容实体（开局 SCP-999）
* MTF 派遣与重收容操作

**主管手记** — SCP-999 已收容且友好。它是你的第一个胜利配额。

**下一步** → 步骤 9：科研

## 步骤 9 · 科技研究

| | |
|---|---|
| **目标** | 打开「科研」标签 |
