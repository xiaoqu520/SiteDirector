## 文档

本手册维护于仓库 `gitbook/` 目录：

* 随 `Directory.Build.props` 的 `GameVersion` 同步
* 主题 CSS 对齐游戏 UI 配色
* 数值以源码为准，发现偏差请提交 issue

2. 推送至 GitHub
