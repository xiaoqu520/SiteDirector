# ⌨️ 快捷键一览

> PC 版完整键位 · Android 见 [移动端章节](../02-getting-started/mobile.md)

---

## 地图与相机

| 按键 | 功能 | 备注 |
|------|------|------|
| `滚轮 ↑↓` | 缩放地图 | 面板打开时先点地图区 |
| `中键拖动` | 平移 | 最直觉的方式 |
| `W` | 向上平移 | 暂停时可用 |
| `A` | 向左平移 | |
| `S` | 向下平移 | |
| `D` | 向右平移 | |
| `↑` `↓` `←` `→` | 平移 | 与 WASD 等效 |
| `左键` | 选中实体 | 房间/人员/SCP → 左下详情 |

## 游戏控制

| 按键 | 功能 | 备注 |
|------|------|------|
| `空格` | 暂停 / 继续 | 教程强制步骤 |
| `Ctrl+S` | 快速存档 | 教程强制步骤 |
| `Esc` | 返回主菜单 | 需确认 |
| `R` | 旋转建造 | 仅建造模式 |
| `Shift` | 修饰键 | 部分拆除操作 |

## 人员

| 操作 | 按键 |
|------|------|
| 选中 | 左键点击人员 |
| 移动 | 右键点击目标格 |
| 解雇 | 右键 → 解雇 |
| 指派岗位 | 详情面板 → 指派 → 点击房间 |

## 建造

| 操作 | 按键 |
|------|------|
| 选择设施 | 建造面板左键 |
| 放置 | 地图左键 |
| 旋转 | `R` |
| 拆除 | 建造面板拆除模式 + 点击 |

## 面板导航

| 操作 | 方式 |
|------|------|
| 切换部门 | 点击左上 Tab |
| 关闭邮件 | 邮件窗口 ✕ 或 Esc |
| 调整 UI 缩放 | 设置面板 |

## 设置页内置提示

游戏内设置面板底部显示：

```
空格 — 暂停 / 继续
Ctrl+S — 快速存档
滚轮 — 缩放地图
中键 / WASD — 平移
Esc — 返回主菜单
每 2 游戏日自动存档
```

## Android 触控速查

| PC | Android |
|----|---------|
| 左键 | 单指轻触 |
| 右键 | 长按 0.5s |
| 平移 | 单指拖动（面板收起） |
| 缩放 | 双指捏合 |
| 暂停 | 地图右下角 II |
| Ctrl+S | 顶栏「存档」 |
| R | 建造模式右下角 R |

## 效率技巧

{% hint style="info" %}
**主管组合技**：`空格` 暂停 → 查阅面板 → 规划建造 → `空格` 继续 → 出问题立刻 `Ctrl+S`。
{% endhint %}

| 场景 | 推荐操作 |
|------|----------|
| 规划布局 | 空格暂停 + 滚轮缩放 |
| breach 应对 | 空格暂停 → 收容面板 → CASSIE |
| 月度结算前 | Ctrl+S → 查看财政 |
| 大规模建造前 | 暂停 → 估算 PowerDraw |
