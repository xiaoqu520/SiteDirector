# 伦理面板

> **v1.8.0** · 处理伦理委员会审查案、申请 D 级测试劳务批准、查看训诫与调查状态。

---

## 打开方式

| 平台 | 入口 |
|------|------|
| PC | 左侧部门导航 → **伦理**（财政与建造之间） |
| Android | 底部「更多」→ **伦理** |

---

## 界面说明

### 合规概览

显示当前 **审计分**、**训诫等级**，以及是否处于：

- 伦理调查期（研究减速）
- 基金会审查（拨款 -8%）
- 伦理办公室是否已建且通电

### D 级测试劳务

1. 点击 **提交 D 级伦理申请** → 开启审查案
2. 在待办案件中回应（提交规程 / 最小化 / 强行启用）
3. 批准后可在本面板 **开关** D 级劳务

设置页中的 D 级开关仅在已批准后可用；未批准时会提示前往伦理面板。

### 待办审查案

- 列表按截止日期排序
- 剩余 ≤2 天 **高亮**
- 点击案件 → 展开描述与回应按钮
- 回应后立即结案并产生审计 / 训诫等效果

---

## 策略提示

| 目标 | 建议 |
|------|------|
| 高审计 playthrough | 优先「提交书面报告」「配合审查」 |
| 快速启用 D 级 | 「最小化方案」中性批准；避免「强行启用」 |
| 降低训诫 | 建造并通电 **伦理审查办公室**，及时回应案件 |

详见 [伦理委员会系统](../06-systems/ethics-committee.md)。
